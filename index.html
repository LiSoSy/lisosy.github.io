The Linux from Source System (LiSoSy) is an automatic custom Linux from source build system inspired from Linux From Scratch by Gerard Beekmans. While other Linux From Scratch automation tools exist, LiSoSy is designed for primarily for enterprise use. Using practices and systems that don't require you to do any special partitioning of existing drives.

Get Started Learn More

Even though the Potabi Foundation is focused on non-Linux operating systems, we felt displeased knowing the extreme number of derivative distribution that take a common base like Ubuntu, slap on a new desktop and a wallpaper, change some default apps, and ship it as a seperate product. As Linux matures in terms of package management, desktop environments, and customization, we felt other projects would benefit from LiSoSy, even if The Potabi Foundation itself finds minimal use beyond experimentation and applications testing. LiSoSy aims to bring a world of possibilities to enterprise and enthusiast Linux users, and make something special to fit the needs of their community, organization, partners, and customers.

Why LiSoSy?

What are the major things Linux from Source System is designed for?

Designed for Independence

Being independent from an upstream is helpful for security. Instead of waiting forever for an upstream patch to be submitted, you can just easily go in and fix it.

Built to be Modular

You customize every last application, when it installs, how it installs, and where it installs. This allows you to have fine control over how things are done.

Made for Everyone

We kept the barrier of entry low, so anyone who has a need for a specifically designed Linux system has the system they need. No gatekeeping non-technical users.
